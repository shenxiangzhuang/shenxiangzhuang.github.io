Exploratory Data Analysis with R读书笔记

发表于 2017-10-06 更新于 2022-03-13 分类于 R 阅读次数： Valine：

启程

这学期开了EDA的课程，课本没用结合程序来进行，所以这里主要依照Exploratory Data Analysis with R这本书来进行EDA编程方面的学习。国庆这两天多花了点时间看完了，这里简单写下笔记。（这书的pdf可以在这里看）

ps:大一的时候学R看的是The R Book，但是看到概率论知识那里就放下了，后来也跑去学Python了orz…这里也当做重新来学R吧。

书上首先介绍了dplyr的使用方法，十分好用（感觉和pandas好像…）。为了加深记忆，用Rmarkdown写了个笔记，发布在rpubs。再到后面主要介绍了EDA的过程和可视化的方法，对ggplot2的介绍间接明了，十分实用。最后还给出了具体的案例分析，结合的很好！

总之，这本书还是比较好的，可以学习EDA的方法，同时还可以作为一个简洁的handbook来备查。
接下来，R的学习就已R in Action为主了，同时，非参数课本上的R程序也打算一直跟下去。

后续

强迫症发作。。。因为一直在用R跟进统计预测与决策，非参数统计的课程，所以对现在的EDA课程也想着用R挨着来实现一下，虽说要写代码的地方不多…承接着上面的EDA WITH R那本书，发现这里 有更为详细的讲解。所以就参考这个，对教材Understanding Robust And Exploratory Data Analysis进行跟进。
