Get help in R

发表于 2017-10-10 更新于 2022-03-13 分类于 R 阅读次数： Valine：

Overview

之前一段时间开始看R in Action， 感觉写的特别好，尤其是一开始对R的帮助文档的使用和基本数据类型的概括，这里做下记录以备忘。首先是多种获取帮助的方式。

Code

demo

书中介绍的第一个TIP就是关于demo的：

To get a sense of what R can do graphically, enter demo()at the command prompt. A sample of the graphs produced is included in figure 1.5. Other demonstrations include demo(Hershey), demo(persp), and demo(image). To see a complete list of demonstrations, enter demo() without parameters.

demo直接给出简洁的代码示例来解释如何使用相关的功能，可以说是十分有帮助的。这里拓展一下：

  • [ ] demo()

直接demo()的话，会返回Demos in package ‘base’:, 也就是base库包含的一些demo。

  • [ ] demo(package = .packages(all.available = TRUE))

通过这条命令，我们可以列出已经安装的包中，所有可用的demo

  • [ ] demo(“zoo-overplot”, package = “zoo”)

a character vector giving the packages to look into for demos, or NULL. By default, all packages in the search path are used. 这里，我们可以通过指定package的值来搜索指定包下的demo。如果不指定，则在所有已导入的包中搜索。

help functions

The vignettes returned by the vignette() function are practical introductory articles provided in PDF format. Not all packages have vignettes

这里的vignettes值得去注意，因为这些包的官方提供的文档，是我们学习包的使用最有效的方法。
