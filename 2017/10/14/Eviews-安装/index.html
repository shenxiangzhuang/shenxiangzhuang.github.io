Windows

关于Eviews9（windows）的安装。

安装文件链接：

链接: https://pan.baidu.com/s/1eSEjcx8 密码: 248h

**
安装过程**

可以参考crack文件夹下的README文件。就是，前面的安装一直默认即可。
等到出现Serial Number的时候，上面输入Demo即可（可能不是必须的，不过最后还是写上这个），下面的是程序自动检测到的你的计算机用户名，不用管。如下：

之后再一路默认即可，无非就是选择下安装的路径这些。直到出现下面这个，询问是否检查更新，禁止即可。

最后再一路默认到最后即可。完成安装后，桌面会出现Eviews的快捷方式。注意这时候直接运行是不可以的，因为还没有激活。只需要找到从百度云下载的文件中，crack文件夹下的Patch文件，运行，点击左侧patch,等待补丁打好退出即可， 如图。

这时候再运行桌面的快捷方式就可以了。

Mac

对于Mac的安装，可以到Eviews官网下载Students Version, 目前版本应该是9.5的。只需要填一些信息就可以了。测试的时候用的学校的教育网邮箱是可以的。也可以用自己的其他邮箱试下，不行再去校内门户的邮件系统申请一个教育网邮箱进行申请。

嗯…还是支持正版好些orz…

Student Version 补充

有些同学说破解文件报毒…我这没有360没报orz…用win10自带的杀毒软件扫了下，是安全的，所以这个一般不用担心。不过还是有的补丁程序没法运行，所以，可以用老师的那个老点的版本，也可以用Student Version（9.5版的，算是最新了，而且Win,Mac都有，而且还免费，力荐啊orz…。）。刚把上面的那个卸了，试了下学生版，很简单几步就可以了。

准备：邮箱

前面说了，测试用的学校的教育网邮箱，是可以的。可以先用自己的其他邮箱试下，不行再去校内门户申请一个。登录进去校内门户，右侧第一个就是邮件系统，进去申请就可以了。

填写注册信息

首先到这里,看到下面的Free Download,点进去，来到注册页面,这里填写好相关的信息即可。

查收邮件

填好后会很快收到邮件（如果收不到的话可能就是自己的邮箱不行，就需要按照前面那样申请教育邮箱了）。邮件会给出下载的链接(Win和Mac都有)，以及序列号。

安装激活

文件下载好直接默认安装就可以，当中不会提示输入序列号。安装好运行，会出现注册的提示，输入邮件的序列号即可（下面的用户名一样可以随便填）。激活后会有注册成功的提示。
