字体大小调整

一般Editor的字体的调整，直接去Setting->Editor改就行了。至于插件的字体调整，如果插件本身无法设置，可以去stylesheet设置。可以Edit->Stylesheet或者Setting->Theme->stylesheet。

比如设置Script运行程序后的输出字体大小为16px，只需要在文件最后加上：

.script-view .line {
font-size: 16px;
}
