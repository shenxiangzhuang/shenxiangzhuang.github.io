### SSR & Proxy
```bash
# set proxy
function setproxy() {
    export http_proxy=socks5://127.0.0.1:1080
    export https_proxy=socks5://127.0.0.1:1080
}

# unset proxy
function unsetproxy() {
    unset http_proxy https_proxy
}
```
